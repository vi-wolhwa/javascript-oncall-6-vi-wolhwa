## 개발 시퀀스

1. [입력] 월, 요일
  pre : delSpace, split(',')
  val : 월:isInRange, 요일:isIncluded

1. [입력] 평일-닉네임, 주말-닉네임
  pre : delSpace, split(',')
  val : 배열: 길이5~35 / 닉네임: 길이5, 중복x

1. [출력] 결과

---

### 비상 근무 배정 규칙
- 기본적으로 순번에 따라 배정
- 연속 2일 불가 => 다음 근무자와 순서 변경

### 기타 요구사항
- 2월은 28일까지만 있다.
- (월,요일)에서 예외 발생 시 (월,요일) 재입력
- 평일/휴일 순번에서 예외 발생 시 평일 순번부터 재입력
- 평일이면서 공휴일인 경우에만 (휴일) 표기
